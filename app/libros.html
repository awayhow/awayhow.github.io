Libros oficiales


Libros oficiales
  • El sistema le permitirá descargar un archivo para su presentación en https://www.registradores.org/

    • Busque y ejecute "Taxor Cloud Impresión Libros Oficiales".
    • Seleccione los libros que desea imprimir.
    • Rellene la información en cada una de las pantallas.
    • El sistema ejecutará el proceso "Valorar stock productos", puede cancelar si lo desea.
    • El sistema ejecutará el proceso "Asignar números de asiento", puede cancelar si lo desea.
    • Al final del proceso se descargará un archivo comprimido en formato Legalia 2.
